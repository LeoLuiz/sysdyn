SysDynModel

Template for Systems Dynamics Model.

Arguments

  • changes: A function that describes how the system changes in each time step.
  • graphics: A table with internal tables describing how the parameters of the model will be drawn in the screen. It can have three values:
  • ...: Any other value that can be used as parameter for the Model.

Usage

Tube = SysDynModel{
    water = 20,
    changes = function(model)
        model.water = model.water - 1
    end
}
